##项目名称
+ 多人聊天室
##项目描述
+ 支持多人同时注册、私聊、群聊和退出功能的网络聊天室
##项目实现
+ Java基础
+ Socket编程
+ 多线程
+ MySQL
+ JDBC
##项目功能
+ 注册
+ 私聊
+ 群聊
+ 退出
